Pointers for Block

  1. Width of the block level element is 100% of the viewport width
  2. Height of a block is not set by default
  3. Height and width can e explicitly set using CSS

Pointers for Inline

  1. Width of the inline level element is dependent on the content
  2. Height of a block is not set by default
  3. Height and width cannot e explicitly set using CSS
